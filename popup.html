Privacy Policy

We store anonymous and encrypted links that will be deleted after 30 days. No personal information is collected or stored.

The encryption happens entirely within your browser using the password you provide. This password is never sent to any server.

The shortened link ID itself is anonymous and does not contain any information about the original content or the user who created it.

The service used for shortening URLs (sorame.danials.space) stores only the anonymous link ID and the encrypted data blob for 30 days, after which it is automatically deleted.
